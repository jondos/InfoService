InfoService

Erweiterte Einstellungen

  • Automatische InfoService-Anfragen durchführen:
    Der JAP versucht an einigen Stellen automatisch eine Verbindung zum InfoService aufzubauen und Abfragen durchzuführen. Wenn dieses Feld nicht markiert ist, unterbleiben diese automatischen Abfragen.

  • Nicht-Anonymisierte Verbindungen zu InfoServices erlauben:
    Ist diese Option aktiviert, kann JAP Verbindungen zu den InfoServices herstellen, auch wenn Sie gerade nicht anonym surfen.
    Wird diese Option nicht aktiviert, können nur über anonymisierte Verbindungen Informationen von den InfoServices abgerufen werden.
    Es wird deshalb empfohlen, diese Option zu aktivieren.

  • Benutze ausschließlich den Standard-InfoService:
    Normalerweise versucht der JAP, die gewünschten Informationen von allen bekannten InfoServices zu erhalten, wenn der Standard-InfoService diese nicht liefern kann. Wenn dieses Feld markiert ist, wird nur der Standard-InfoService abgefragt.
    Dies wird NICHT empfohlen.
    Wird diese Option NICHT aktiviert, kann zusätzlich folgende Einstellung vorgenommen werden:

    • Anzahl redundant abzufragender InfoServices:
      Legen Sie fest, von wievielen InfoServices Informationen geholt werden sollen. Legen Sie z.B. als Anzahl "3" fest, so wird JAP immer versuchen, 3 InfoServices zu erreichen. Fällt einer aus, wird versucht einen anderen als Ersatz zu erreichen.
      Je mehr InfoServices Sie festlegen, desto robuster wird JAP gegen Störungen in den InfoServices, desto länger dauert die Abfrage aber auch.
