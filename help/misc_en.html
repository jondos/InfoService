Miscellaneous

If you click Settings or Change on the register card Configuration, a new window is opened which contains the register card Miscellaneous among other configuration opportunities.

On the card Miscellaneous, you can define the language used by the operating platform and change the Look-and-Feel in future versions.

The setting opportunities are currently restricted to the Debbugging. In this place you can configure the extent to which status and error messages will be displayed on the so-called console (e.g. in the DOS window). If there are any problems occurring with the JAP, you can probably locate them according to these error messages. If you are not able to find the solution and therefore ask us for help, please include the corresponding messages in your e-mail.

You can choose to receive the messages referring to a certain area by activating the corresponding check. At Level, you can configure how important the displayed messages have to be. If you choose level 0, all messages will be shown. Level 7 leads to displaying only messages which refer to serious problems that prevent the service from working properly.

Please read also the page Trouble Shooting.
