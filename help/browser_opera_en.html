JAP Setup for Opera

To set up JAP for Opera, do the following steps:

1.) From the "Extras" menu, choose the submenu "Settings". In the new window that opens, choose the category "Network/Connections"



2.) Now you can change the proxy server settings. Click on the "Proxy server" button. A new window opens where the following settings need to be changed:
Click on the checkboxes for HTTP, HTTPS, and FTP. In each of the fields, enter the server address "127.0.0.1". Enter "4001" for the port.



Note: If you set a different listener port in JAP, you'll have to enter this port here too, rather than 4001.
