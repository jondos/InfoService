Listener Port Number

To make sure that the JAP is able to anonymize the communication with the internet, it has to be interposed between the browser and the server in the internet. Such an in-between station is generally called proxy. Most applications that communicate with the internet can direct their connections via such a proxy. The necessary statements for this are: The name of the computer (host) on which the proxy is operated, and the port number via which the proxy receives requests. The default port number of the JAP is 4001. This configuration can be stuck to in nearly all cases.

Essential configurations at the web browser

For making sure that your web browser sends its requests to the web actually via the JAP, you have to configure it in a way that it uses the JAP as a proxy. Generally, you use the following procedure:

    1.    Identify the proxy configuration in the browser settings.
    2.    Activate the proxy modus (use proxy server).
    3.    Enter 127.0.0.1  into the field Host, and then enter the port number configured in the JAP
           (Standard: 4001) into the field Port.

Further support concerning the necessary Settings of the Internet Explorer and the Netscape Navigator.

Changing the port number in the JAP

In very rare cases it can become necessary to change the port number, e.g. if the default port is already being used by another application in your system. On the register card Configuration the current port number of the JAP is being displayed in the top frame, which is called Listener Port Number. Below this, the current status of usage is being displayed. If the port is actually in use, the display shows Status: activ.

For changing the port number, proceed as follows:

    1.    Click Settings next to the port number display.
    2.    Enter the desired port number.
    3.    Click OK.
    4.    Check if the desired port number can be used (Status: aktiv must be displayed).
           If this is not the case, try another port number.
    5.    Change the configurations of your web browser respectively.
