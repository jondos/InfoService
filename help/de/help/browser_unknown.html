Proxy Einstellungen für Browser: Andere Browser

Theoretisch ist es möglich JonDo für jeden Browser einzurichten, der Surfen über einen HTTP-Proxy unterstützt. Dazu müssen Sie in den Einstellungen ihres Browsers nach einer Konfigurationsmöglichkeit für einen Proxyserver suchen und folgendes eintragen :
  • Adresse des HTTP-Proxy : 127.0.0.1
  • Port des HTTP-Proxy : 4001

Normalerweise können Sie bei diesen Einstellungen auch Ausnahmen festlegen, wann JonDo nicht verwendet werden soll.
Beachten Sie bitte, dass solche Ausnahmen flexibler bei einem Filterproxy wie Proxomitron vorgenommen werden können.


Hinweis: Falls Sie im JonDo einen anderen Listenerport eingestellt haben müssen Sie diesen anstatt 4001 eintragen.
