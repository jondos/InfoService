User Interface

In this dialog, you can make changes to the user interface appearance, language, and message displaying behavior of JAP.

Look-and-Feel Settings

  • Select Look-and-Feel:
    JAP supports various forms of graphic appearance for the program windows and user interface elements. Depending on the operating systems, several options are available.

    • LookAndFeel
      Choose your preferred look and feel from the dropdown list. These changes require an instantaneous restart of JAP to take effect.
      You may add new look and feel options by downloading some from the internet and importing them to JAP. You need no packages created specifically for JAP, instead you may use every look and feel package suitable for java.
      Such packages can be found at www.javootoo.com. After downloading a package, klick Import to add this option.

    • Select language:
      Here you can change the language in which the user interface elements appear.
      Here, too, a restart of JAP is needed for the changes to take effect.

    • Font size
      Here you can change the font size of the user interface elements.
      These changes have immediate effect.

    • Remember location of JAP:
      If you put a check in this box, the JAP program window will appear at the previous position when you close and restart JAP.

Default view of JAP

  • Normal:
    In normal mode, all options in the main window and in the configuration dialog will be present.

  • Simplified:
    In simplified mode, some advanced options will not be shown. The functioning of JAP is not changed by this, however.

Programm Start

  • After start of JAP...:
    If you activate this option, there are two further possible options:

    • ...show the minimal view:
      This option minimizes the main JAP window immediately after starting.

    • ...move JAP into the Systray:
      If this option is chosen, the main JAP window moves to the Windows Systray immediately after starting.

Program shutdown

  • Show warning when closing JAP
    When checked, JAP displays a warning which says that your browser's proxy settings need to be reconfigured in order to continue surfing the web.
