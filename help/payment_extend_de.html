Bezahlung: Erweiterte Einstellungen

Dieses Panel steht nur in der Expertenansicht zur Verfügung.

Um zu diesem Einstellungsfenster zu kommen, klicken Sie im Hauptfenster von JAP auf Einstellungen, wählen anschließend aus der linken Liste den Eintrag Bezahlung und öffnen anschließend den Tab Erweiterte Einstellungen
. In diesem Dialog können Sie erweiterte Einstellungen für die Bezahlung vornehmen.

  • Bestätigungsdialog vor der Benutzung eines kostenpflichtigen Dienstes anzeigen
    Wenn Sie diese Option aktivieren, wird ihnen ein Dialog zur Bestätigung angezeigt bevor JAP eine kostenpflichtige Kaskade benutzt. Das hat auch zur Folge, dass JAP sich nicht automatisch mit einer Bezahlkaskade verbinden kann.

  • Nicht-anonymisierte Verbindungen zu Bezahlinstanzen erlauben
    Ist diese Option aktiviert, kann JAP Informationen über Ihren Kontostand abfragen, auch wenn Sie gerade nicht anonym surfen.
    Wird diese Option nicht aktiviert, können nur über eine anonymisierte Verbindung Kontodaten abgerufen werden.
    Es wird deshalb empfohlen, diese Option zu aktivieren.

  • Fehlermeldungen der Abrechnungsinstanz melden (Nur in der Entwicklerversion verfügbar)
    Wenn diese Option aktiviert ist, werden Fehlermeldungen bezüglich der Abrechnungsinstanz angezeigt. Es wird empfohlen, diese Option zu aktivieren.
    In den stabilen Versionen ist dies immer aktiviert.

  • Kontostand automatisch aktualisieren
    Wenn Sie diese Option aktivieren, versucht JAP in regelmäßigen Zeitabständen Ihre Kontoinformationen zu aktualisieren.
    Wenn Sie dies nicht erlauben, wird in JAP nicht automatisch der aktuelle Kontostand angezeigt. Das Konto wird bei der Bezahlinstanz dennoch korrekt geführt.
