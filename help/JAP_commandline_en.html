Command line arguments for JAP

JAP can be launched from the command line. Several commands can be specified on startup.
    Commands (long version, short version respective):

  • --help, -h

    Shows a help message.

  • --console

    JAP will only be running on console. This means that no graphical user interface will be shown.
    In order to quit JAP, type "exit" and press Enter.
    In order to save configuration, type "save" and press Enter.

  • --minimized, -m

    JAP minimizes it's window during startup.

  • --noSplash, -s

    Suppresses the splash screen at JAP startup. Starting JAP will then seem to take a little bit longer, as it takes some time until the JAP main window is initialised. The spash screen's function is to bridge this time gap.

  • --version, -v

    JAP will print version information for JAP und Java onto the console window.

  • --config {Filename}, -c {Filename}

    JAP will use this configuration file.

Example

In order to start JAP minimized, open a console window, switch to JAP's directory and enter the following command:

java -jar JAP.jar -minimized

You may also use these arguments in Windows shortcuts. Just click on the JAP shortcut with the right mouse button, choose Settings and edit the shortcut as follows:

Target: <Path to JAP.jar>\jap.exe -minimized
Execute in: <Path to JAP.jar>

Replace hereby <Path to JAP.jar> with the path where the JAP.jar and the japdll.dll are located (typically C:\Programme\Jap).
