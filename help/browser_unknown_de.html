Einstellungen für verschiedene Browser

JAP für andere Browser

Theoretisch ist es möglich JAP für jeden Browser einzurichten, der Surfen über einen HTTP-Proxy unterstützt. Dazu müssen Sie in den Einstellungen ihres Browsers nach einer Konfigurationsmöglichkeit für einen Proxyserver suchen und folgendes eintragen :

  • Adresse des HTTP-Proxy : 127.0.0.1
  • Port des HTTP-Proxy : 4001

Hinweis: Falls Sie im JAP einen anderen Listenerport eingestellt haben müssen Sie diesen anstatt 4001 eintragen.
