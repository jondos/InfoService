JAP Setup for Other Browsers

It's theoretically possible to set up JAP for any web browser that supports surfing over an HTTP proxy. You simply have to look in the settings menu of your browser for the possibility of configuring a proxy server and enter the following:

  • HTTP proxy address: 127.0.0.1
  • HTTP proxy port: 4001

Note: If you set a different listener port in JAP, you'll have to enter this port here too, rather than 4001.
