Navegar Anonimamente

Depois de ter descarregado e iniciado o Java Proxy Anon, deve seguir os seguintes passos para poder navegar na web anonimamente.

  • Ligue o seu computador à Internet.
  • Active a ligação anónima através da  interface do utilizador.
  • Configure o seu browser para usar o JAP como um HTTP proxy. introduza os seguintes dados nas definições de proxy:
       Anfitrião: 127.0.0.1
       Porta: 4001
    Uma mais detalhada descrição pode ser encontrada aqui.

Agora pode navegar anonimamente. Para sua protecção, também deve desactivar os cookies no seu browser. De outra forma, os operadores de servidores web podem ser capazes de reconstruir o seu comportamento de navegação, através do envio e recepção de cookies. Se não revelar a sua identidade directamente, não é possível ao servidor descobri-la.

Nota: depois de a configuração do seu browser ser mudada, todas as suas ligações web serão feitas através do JAP. Para navegar anonimamente, tem de iniciar o JAP para além do seu browser e activar a ligação anónima.  Se quiser deixar de navegar anonimamente, deve repor as configurações iniciais de proxy do seu browser.
