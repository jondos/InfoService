Update der JAPDll

Nach einem JAP update kann es notwendig sein, dass auch eine neue Version der japdll.dll verwendet wird. Bei dem Versuch die japdll.dll zu aktualisieren kann es dabei zu folgenden Problemen kommen:
(Hinweis: Wird die dll beim JAP-Start nicht geladen, dann wird auch nicht versucht diese zu aktualisieren. Die dll ist zum reibenungslosen Betrieb von JAP nicht erforderlich, bietet aber einige graphische Vorzüge. z.B das Minimieren in die Symbolleiste)
  • Eine vorhandene japdll.dll konnte nicht aktualisiert werden, da die benötigten Schreibrechte fehlen.
    Lösung: Stellen sie sicher, dass Sie die benötigten Schreibrechte besitzen. Führen Sie z.B. zum Zweck des Updates den JAP (einmalig) als Administrator aus, oder lassen Sie einen Administrator die japdll.dll aktualisieren.
  • Es wurde beim JAP-Start eine japdll.dll von einem unbekannten Ort geladen. Vermutlich liegt eine japdll.dll in Ihrem Windows-System-Verzeichnis oder an einer anderen Stelle im PATH. Der Ort konnte jedoch nicht ermittelt werden. Es wird versucht die japdll.dll in dem Verzeichnis zu aktualisieren, in dem auch die JAP.jar liegt. Wenn an dieser Stelle keine japdll.dll liegt, dann schlägt das Update fehl.
    Lösung: Ersetzen sie die vorhandene japdll.dll auf Ihrem System durch die neue Version der japdll.dll. Oder löschen Sie die vorhandene japdll.dll auf Ihrem System und speichern Sie die neue Version der japdll.dll in das Verzeichnis, in dem die JAP.jar liegt.
  • Es wurde beim JAP-Start eine JAPDll von einem unbekannten Ort geladen. Vermutlich liegt eine japdll.dll in Ihrem Windows-System-Verzeichnis oder an einer anderen Stelle im PATH. Der Ort konnte jedoch nicht ermittelt werden. Es existiert eine japdll.dll in dem Verzeichnis in dem die JAP.jar liegt. Das Update dieser Datei verläuft zunächst erfolgreich und die japdll.dll wird durch die neue Version ersetzt. Beim JAP-Start wird jedoch nicht diese (neue) japdll.dll geladen, sondern wieder eine alte Version von einem unbekannten Ort.
    Lösung: Ersetzen sie die vorhandene japdll.dll auf Ihrem System durch die neue Version der japdll.dll. Oder löschen Sie die vorhandene japdll.dll auf Ihrem System. Beim nächsten JAP-Start wird dann die (neue) japdll.dll Version aus dem JAP.jar Verzeichnis geladen.
