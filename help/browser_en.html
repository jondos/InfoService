Proxy Configuration

Configurations for the Internet Explorer (PC)

    1.    In the menu Extras please select Options.
    2.    Select the register card Connections.
    3.    If you are connected to the internet via a local network, click the button LAN configurations.
           If you connect to the web via analogous or ISDN line, select your internet connection in the
           VPN settings and click the button Configurations.
    4.    Activate the option use proxy server.
    5.    Enter 127.0.0.1  into the address field and then enter the port number configured in the JAP
           (standard: 4001) into the connection field.
    6.    Click “OK” until all configuration windows are closed.

Configurations for the Netscape Navigator (PC)

    1.    Select Configuration in the Edit menu.
    2.    On the left side you can see the possibilities for configuration. Click + next to the Advanced,
           entry to view the corresponding branch of the tree.
    3.    Click Proxies.
    4.    Select Manual proxy configuration on the right side of the window and click the View button.
    5.    Enter 127.0.0.1  in the address field in the lines HTTP, FTP, GOPHER, WAIS and Security (HTTPS), then enter the port number
           configured in the JAP in the field Port (Standard: 4001).
    6.    Click OK until all configuration windows are closed.

Configurations for Apple Macintosh

    1.    Select the control item Internet.
    2.    Click Advanced.
    3.    Select Firewall  from the options list.
    4.    Select Use proxy server and then enter the proxy address 127.0.0.1 at Web Proxy and
           FTP Proxy and the port number configured in the JAP (Standard: 4001).
    5.    Press Command-Q and confirm the changes you made.
