Proxy/Firewall

Alguns utilizadores não podem usar o serviço de anonimato directamente. É o caso se estiverem bloqueados por uma firewall e portanto não se podem ligar à primeira mistura. Além disso, alguns fornecedores de Internet requerem o uso obrigatório de um proxy.

Se está certo que é forçado a usar um proxy obrigatório, pode mudar aqui a configuração da ligação de proxy.

Proxy/Firewall

  • Sou forçado a usar um proxy
    Seleccione esta caixa para mudar as configurações de proxy obrigatório.

  • Tipo
    Seleccione o tipo de proxy. Os tipos possíveis são "Socks-Proxy" e "HTTP/HTTPS-Proxy".

  • Anfitrião
    Introduza o nome de anfitrião ou o endereço IP do seu proxy.

  • Porta
    Introduza o número da porta na qual o seu proxy recebe ligações.

  • O Proxy requer autorização
    Seleccione esta caixa se tiver que se "logar" no seu proxy.

  • ID de Utilizador
    Introduza o seu nome de utilizador para se ligar ao proxy. A palavra-passe será automaticamente pedida quando se ligar ao proxy pela primeira vez.

Se tiver uma firewall no seu computador, configure-a de modo a que o JAP possa ligar à Internet, especialmente aos endereços do InfoService e da primeira mistura.
