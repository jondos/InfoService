Einspielen von Updates

In diesem Dialog können Sie Einstellungen vornehmen, die die grafische Erscheinung, die Sprache und das Anzeigeverhalten nach dem Starten von JAP festlegen.

Installierte Version

  • Version:
    Hier sehen Sie die Versionsnummer der derzeit auf Ihrem System installierten Version von JAP.

  • Datum:
    Dieses Feld zeigt Ihnen den Zeitstempel der momentan installierten Version.

  • Art:
    Hier wird angezeigt, um welche Art es sich bei der installierten Version handelt. Es gibt zwei verschiedene Versionsarten: Die "stabile Version" wurde ausgiebig getestet und enthält nur Funktionen, von denen angenommen wird, dass Sie keine Fehler enthalten. Die "Entwicklerversion" enthält alle derzeit verfügbaren Funktionen, auch wenn diese nur teilweise implementiert oder fehlerhaft sind.
    Für Normalbenutzer wird empfohlen, die "stabile Version" zu verwenden.

Aktuellste Version

In dieser Gruppe sind die gleichen Felder wie in der Gruppe "Installierte Version" vorhanden. Die Informationen in den Feldern entsprechen den Informationen in der Gruppe "Installierte Version". Die hier angezeigten Informationen beziehen sich aber nicht auf die auf Ihrem System installierte Version, sondern auf die Version von JAP, die momentan herunter geladen werden kann.
Mit Hilfe der Auswahlbox "Art" können Sie festlegen, ob Sie Informationen über die "Stabile Version" oder über die "Entwicklerversion" wünschen.

Info

In diesem Feld werden Ihnen Informationen über den Fortschritt beim Installieren eines Updates angezeigt.

Weitere Bedienelemente

  • Prüfe, ob neue Updates vorhanden sind:
    Wenn Sie diesen Knopf anklicken wird geprüft, ob sich aus dem Internet eine neuere Version als die derzeit bei Ihnen installierte herunterladen lässt.

  • Upgrade:
    Hiermit können Sie die neueste Version von JAP aus dem Internet laden und auf Ihrem System installieren.
