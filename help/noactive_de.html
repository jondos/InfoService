Aktive Inhalte

Webseiten können Inhalte anbieten, die Ihre Identität aufdecken können, obwohl sie mit JAP anonym surfen. Das kann z.B. über JavaScript oder auch Java geschehen.
Teilweise bieten Browser selbst komfortable Möglichkeiten, die Verwendung von solchen Inhalten zu unterbinden, teilweise kommen dafür eher Zusatzprogramme in Frage.
