InfoService

Advanced Settings

This panel is only available in Expert View.

To get to this panel, press button Settings in JAP's main window, then choose the entry InfoService from the list to the left, then open the tab Advanced Settings.

  • Enable automatic InfoService Requests
    JAP occassionally attempts to automatically connect to the InfoService to request information. If this checkbox is activated, these automatic connections are also activated.

  • Allow non-anonymous connections to InfoServices
    If activated, JAP can connect to InfoServices even if you have no anonymized connection currently established.
    If this option is deactivated, information from InfoServices may be gathered through anonymized connections only.
    Thus, it is recommended to activate this option, since no other InfoService will be used if this one is offline.

  • Do redundant requests on different InfoServices:
    It is recommended to activate this option.
    Define how many InfoServices will be queried. If you set this value to "3", for example, JAP will always try to reach 3 InfoServices. Should one of them appear to be offline, JAP will try to reach a substitute.
    The higher the value, the more fail safe JAP will become, the more time these queries will consume, however.
