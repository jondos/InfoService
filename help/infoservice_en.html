Info Service

General Issues

The Info Service is used for provide the JAP with necessary information on the anonymity service. At the moment this includes:

  • The current JAP version code for automated updates.
  • The address of the first mix.
  • Information on the mix cascade, e.g. the public keys of the mixes.
  • Information on the service’s capacity and the therefore achieved state of anonymity.
Configuration

In the register card Configuration, the URL of the info service is to be entered at Info Service. For changing the URL, hit the button Change and enter the new URL incl. port number.

Please notice that the correct URL is already provided. Altering the URL may disable the JAP of using the anonymity service.
