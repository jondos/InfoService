Services

General

  • Assignment of internet services to anonymizer services
    This overview shows which internet service is assigned to which anonymizer service. Future releases of JAP will allow you to set your own assignments. Meanwhile, this is merely for your information.

  • Deny all data traffic without call back if anonymous mode is switched off
    If this box is checked, JAP will not transfer non-anonymous traffic. Your browser will show a hint which says that browsing the web is currently not possible.
    If you do NOT check this box, JAP will show a warning window which has the option to allow non-anonymous browsing for a short amount of time.
    More information

  • Auto connect after program start
    If this box is checked, JAP automatically connects to the anonymization service when JAP is started.

  • Auto re-connect after connection to mix was lost
    Check this box if you want JAP to automatically reconnect to the anonymization service if the connection is lost.

  • Automatically change Mix cascades
    Checking this box will give you two more options:

    • Equally choose all cascades
      With this option activated, JAP will choose a cascade by chance. During anonymous browsing, however, cascades will only be switched if the active cascade is offline.

    • Only choose payment cascades
      Similar to the first option, with the distinction that the random cascade will exclusively be chosen from payment cascades.

  • Send dummy packet every x seconds
    Here you can decide if fake packets without any content should be sent and if so, how often. This simulates continuous communication for any potential observers.
