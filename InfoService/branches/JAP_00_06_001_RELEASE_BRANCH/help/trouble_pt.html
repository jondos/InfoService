Problemas

Erro de Ligação ao InfoService

Quando tenta activar o acesso anónimo, uma janela pop-up com a seguinte mensagem aparece: Erro ao contactar o serviço de informação JAP.

  • Verifique as configurações do Infoservice. Os seguintes valores têm de estar definidos: Anfitrião: infoservice.inf.tu-dresden.de, Porta: 6543. Se estes valores não estiverem correctos, por favor, corrija-os.
  • Verifique se a lua ligação à internet funciona correctamente.
  • Verifique se o seu computador está protegido por firewall. Se for o caso, peça ao seu administrador de sistema para libertar a porta 6543. Infelizmente, sem isto o JAP não funciona. Comunicação directa com o InfoService ou a primeira mistura via HTTP proxy está planeado para futuras versões, o que resolverá o problema.
  • Se nenhum dos tópicos acima resolverem o seu problema, a responsabilidade não deve ser sua. Seja paciente e tente mais tarde.

Erro de Ligação à primeira Mistura


Quando tenta activar o acesso anónimo, uma janela pop-up com a seguinte mensagem aparece: Erro ao ligar à primeira Mistura!.

  •  Verifique as configurações da primeira mistura, escolha uma mistura diferente ou recarregue o URL do InfoService.
  • Verifique se o seu computador está protegido por firewall. Se for o caso, peça ao seu administrador de sistema para libertar a porta fornecida no endereço da mistura. Infelizmente, se não puder fazer isto, não pode usar o JAP. Comunicação directa com o InfoService ou a primeira mistura via HTTP proxy está planeado para futuras versões, o que resolverá o problema.
  • Se nenhum dos tópicos acima resolverem o seu problema, a responsabilidade não deve ser sua. Seja paciente e tente mais tarde.

O browser não funciona correctamente ou não usa o JAP para os pedidos à internet 

  • Verifique se o seu browser  está configurado correctamente.
  • Verifique se activou o  acesso anónimo.
  • Verifique se a lua ligação à internet funciona correctamente.

Suporte Via E-Mail

Se tiver um problema especial, tente fazer uma aproximação usando as  mensagens de depuração da consola. Se não conseguir fazer isto, vá à página web do JAP http://anon.inf.tu-dresden.de para ver se a solução para o seu problema lá está mencionada. Se não estiver, envie-nos um e-mail para o seguinte endereço: jap@inf.tu-dresden.de

Esse e-mail deve ter a seguinte informação:

  • Uma descrição detalhada do erro e da(s) situação(ões) em que ocorre. 
  • Uma cópia das mensagens de erro mostradas na consola. Para obter as mensagens de erro apropriadas, deve activar todas as verificações relativas à depuração na área  Depuração /Config e definir o nível da mensagem para "Depuração".
  • Uma descrição da sua ligação à internet (ISP, rede local, firewall).
  • Um endereço de e-mail para o qual possamos responder.

Seja paciente. Se muita gente tiver os mesmos problemas, a resposta pode demorar um pouco.
