Certificate Authorities

  • I only want to use anonymous servers that have been certified by the following authorities:
    This list contains all certification authorities that you trust.

  • Import...
    With this button, you can add certification authorities to the list of trusted authorities. The appropriate certificate should be chosen in the dialog field that appears.

  • Remove
    This button removes an entry from the list.

  • Disable
    By clicking this button, you can deactivate an entry without removing it from the list.

  • Certificate Authority Information
    Here you see detailed information about the currently selected certification authority.
