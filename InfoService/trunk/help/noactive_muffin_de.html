Verwendung eines Filterproxys

Muffin

Muffin ist OpenSource und wie JAP in Java programmiert.

  • Installation

    Laden Sie sich Muffin von dieser Seite herunter. Verwenden Sie bei den Downloads die Datei mit der Endung .jar oder verwenden Sie direkt diesen Link zum Download.
    Sie können Muffin mit einem Doppelklick auf die heruntergeladene Datei öffnen oder über den Befehl java -jar Dateiname.jar wenn Sie sich im Verzeichnis von Muffin befinden.

  • Einrichtung

    Das Fenster von Muffin öffnet sich. Öffnen Sie im Menü Edit -> Options.


    Geben Sie für HTTP Proxy und HTTPS Proxy jeweils "127.0.0.1" und für den zugehörigen Port "4001" ein.
    Hinweis: Falls Sie im JAP einen anderen Listenerport eingestellt haben müssen Sie diesen anstatt 4001 eintragen.
    Lassen Sie alle anderen Felder unverändert. Bestätigen Sie mit durch Klicken von Apply, Save und Close. Ohne Apply überdauern die Einstellungen einen Neustart von Muffin nicht.
    Durch diese Einstellung holt sich Muffin die Daten aus dem Internet nicht direkt, sondern vom JAP.

  • Filter

    Jetzt müssen die Filter aktiviert werden. Öffnen Sie im Menü Edit -> Filters.


    Wählen Sie von den Supported Filters den Eintrag NoCode und klicken Sie Enable. Wiederholen Sie den Schritt für den Eintrag Referer. Dadurch sind beide Einträge jetzt unter Enabled Filters zu sehen. Es wird empfohlen, beim Filter NoCode die Standardeinstellungen zu belassen. Sollten Sie dennoch Änderungen vornehmen wollen, bearbeiten Sie den Filter durch einen Klick auf Preferences und bestätigen Sie mit Apply, Save und Close.
    Dadurch blockiert Muffin unter anderem Java, JavaScript, VBScript und andere Scriptsprachen und sendet keinen Referer. Bestätigen Sie den Dialog mit Save und Close.

  • Browser

    Als letzter Schritt muss der Browser so eingerichtet werden, dass er die Daten nicht von JAP, sondern von Muffin bezieht. Das Vorgehen ist das selbe wie hier beschrieben, nur verwenden Sie diesmal den Port von Muffin, das ist standardmässig 51966.
    Somit besteht folgende Kette: Browser <-> Muffin <-> JAP <-> Internet.
