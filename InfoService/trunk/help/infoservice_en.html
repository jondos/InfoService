InfoService

The Info Service is used to provide JAP with necessary information about the anonymity service. At the moment this includes:

  • The current JAP version number to allow automatic updates.
  • The address of the first mix.
  • Information on the mix cascade, e.g. the public keys of the mixes.
  • Information on the service's capacity and the level of anonymity achieved by JAP.
Please note that the correct information for the InfoService is already set by default. Changes to these settings could cause JAP to no longer find the anonymization service.

Configuration

On this page, you can see the InfoServices that are known to your JAP client. The currently preferred InfoService appears in bold. You can take the following actions:

  • Download list...
    Here you can download a new list of InfoServices from the internet.

  • Set as default
    By clicking this button, the selected InfoService will be chosen as the preferred InfoService

  • Add...
    When you choose this action, an additional dialog area for manually adding an InfoService appears below.

    • InfoService host
      Enter the host name or IP address of the InfoService that you want to add here.

    • InfoService port
      Enter the port of the new InfoService here.

    • InfoService name
      If you would like, you can enter a name for the InfoService here. The new InfoService then appears under this name in the list of known InfoServices. If this field is left empty, the InfoService is listed as a combination of host name and port.

    • Add
      Click this button to confirm your entry and insert the new InfoService.

    • Cancel
      With this button, the entries for the new InfoService will be discarded and you will return to the previous window.

  • Remove
    When you click on this button, the selected InfoService will be removed from the list of known InfoServices.

Advanced Settings

  • Timeout for InfoService and Update Communication
    Enter a number between 1 and 60 here. This number indicates the number of seconds that JAP should wait for a response to a request sent to the InfoService. If a response is not received within the set time, the InfoService will be considered unavailable.

  • Disable automatic InfoService requests
    JAP occassionally attempts to automatically connect to the InfoService to request information. If this checkbox is activated, these automatic connections are deactivated.

  • Use only the default InfoService
    Normally, JAP attempts to obtain information from all known InfoServices if the standard InfoService cannot deliver the information. When this checkbox is activated, only the standard InfoService will be connected to.
