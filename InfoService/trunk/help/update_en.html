JAP Update

Installed Version

  • Version
    Here you see the version number of the JAP program that is installed on your system.

  • Date
    This field shows the timestamp of the currently installed version.

  • Type
    The type of the currently installed version is displayed here. There are two possible types: The "Release Version" has been thoroughly tested and includes only the functions that should have no bugs or conflicts. The "Development Version" includes all functions that are available, even if they are only partially implemented or contain bugs or mistakes.
    For most users, we recommend the "Release Version".

Latest Version

This area contains the same fields as the "Installed Version" area. The information in these fields also corresponds to the fields in the "Installed Version" area. However, the information does not refer to the software currently installed on your system but rather refers to the JAP version that can currently be downloaded. Using the drop-down menu "Type", you can choose whether the information for the "Release Version" or "Development Version" is displayed.

Optionen

  • Allow non-anonymous update
    It is recommended to activate this option. Thereby JAP can perform updates even if you have no anonymized connection currently established.
    If this option is deactivated, updates may be done via anonymized connections only.

  • Show information window if new JAP version is available
    If this option is active, an information window will be shown whenever a new version of JAP is available.
    Even if this option is not active, there will appear a small hint on the main window anyway.
    JAP checks for new versions on startup.

  • Show information window if new version of Java (JRE)is available
    If this option is active, an information window will be shown whenever a new version of Java is available.
    JAP checks for new versions on startup. Upgrading JRE has to be done manually.

Info

This field displays information about the update's installation progress as the update is being installed.
Furthermore, it shows the result of checking for a new version.

Other Elements

  • Check for new updates
    Click this button to check if a newer version than the currently installed version is available for download.

  • Upgrade...
    With this button, you can download the latest version of JAP from the internet and install it on your system.
