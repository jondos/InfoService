Anonymity

How to use the anonymizing service

On the register card Configuration, the settings and the current status of the JAP concerning the connection to the anon service is being displayed the within option Anonymity. If the check for Activate anonymous Access is activated, the JAP directs all requests via the anon service. Directly below this, the URL of the first mix is being displayed. Additionally, there is a status display that provides information about the service’s usability. If you want to surf anonymously, it must show Status: active.

Please note that there must be a connection to the web before the activation, because the JAP establishes a permanent connection to the first mix. Since this connection still remains active even if you don’t send any requests and since status information is exchanged on a regular basis, you should deactivate the anonymous access before you disconnect from the internet.

Please note additionally that the automatic disconnection does not work after a certain time of inactivity if the anonymous access is activated. This is due to the system’s inability to realize the inactivity since the JAP permanently orders information.

Configuration

You can change the URL of the anon service if you want to use a different mix cascade. For this please click on the button Change.

Now there are three possibilities to call another anon service:

  • You can ask Info Service to update the locally stored list of anon services. For this, choose  Download and click the load button. This procedure updates the local database.
  • From the local database you can select an anon service by choosing Select in local Database and clicking on the name of one of the anon service in the opened dropdown menu.
  • Apart from this you have the opportunity to enter the URL of the anon service manually.
Manually entering the anon service as well as changing the URL makes sense only if there are alternative operators of mix cascades who work with this protocol. For using the anon service, only the first two settings should be used.
