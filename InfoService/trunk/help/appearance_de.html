Verändern des Erscheinungsbildes

Um zu diesem Einstellungsfenster zu kommen, klicken Sie im Hauptfenster von JAP auf Einstellungen und wählen anschließend aus der linken Liste den Eintrag Erscheinungsbild.
In diesem Dialog können Sie Einstellungen vornehmen, die die grafische Erscheinung, die Sprache und das Anzeigeverhalten nach dem Starten von JAP festlegen.

Erscheinungsbild

  • Erscheinungsbild:
    JAP unterstützt verschiedene Formen der Darstellung von Fenstern und Bedienelementen.
    • Sprache
      Hier können Sie die Sprache einstellen, in welcher die Bedienelemente erscheinen sollen.
      Auch hier ist ein sofortiger Neustart von JAP erforderlich um die Änderung zu übernehmen.

    • LookAndFeel
      Wählen Sie hier ihr bevorzugtes LookAndFeel aus der Liste aus. Die Änderungen erfordern einen sofortigen Neustart von JAP.
      Sie können neue LookAndFeel-Optionen hinzufügen, indem Sie sich aus dem Internet neue Pakete herunterladen und in JAP importieren. Diese sind nicht JAP-spezifisch, Sie können alle Pakete für Java (Swing) verwenden.
      Solche Pakete finden Sie z.B. auf www.javootoo.com. Laden Sie ein solches Paket auf ihren Computer und klicken Sie auf importieren um die neue Option hinzuzufügen.

      • Importieren
        Ein Klick darauf öffnet ein Auswahlfenster. Navigieren Sie zum Speicherort des neuen LookAndFeel Pakets und wählen Sie die Datei mit der Endung ".jar".

      • Entfernen
        Damit können Sie ein LookAndFeel löschen, wobei dies nur bei solchen funktioniert, die sie selbst herunter geladen haben. Das aktuell ausgewählte LookAndFeel können Sie ebenfalls nicht löschen. Das LookAndFeel bleibt dabei bis zu einem Neustart von JAP im Speicher.

    • Schriftgrösse
      Hier können Sie die Schriftgrösse der Bedienelemente einstellen.
      Diese Änderungen werden sofort aktiv.

    • Fensterposition von JAP merken:
      Wenn Sie bei diesem Kästchen einen Haken setzen, dann erscheint das Programmfenster von JAP bei jedem Start an der gleichen Stelle, an der es sich beim zuvorigen Beenden befand.

Ansicht von JAP

  • Experten-Ansicht:
    In der Experten-Ansicht werden alle Optionen im Hauptfenster sowie in den Konfigurationsdialogen angezeigt.

  • Vereinfachte Ansicht:
    In der vereinfachten Ansicht werden einige fortgeschrittene Optionen nicht angezeigt. Die Grundfunktionalität des JAP ändert sich dadurch nicht.
Diese Änderungen erfordern einen sofortigen Neustart von JAP.

Programmstart

  • Gehe nach dem Start...:
    Wenn Sie diese Option aktivieren, haben Sie zwei weitere Auswahlmöglichkeiten:

    • ...in die Minimalansicht:
      Diese Option minimiert das Hauptfenster des JAP sofort nach dem Start.

    • ...in den Systray:
      Wenn diese Einstellung gewählt wurde, verschwindet das Hauptfenster des JAP gleich nach dem Start im Systray von Windows.

Programmende

  • Warnung beim Beenden von JAP
    Diese Option zeigt eine Warnung beim Beenden von JAP um darauf aufmerksam zu machen, dass die Proxykonfiguration im Browser geändert werden muss, wenn man weitersurfen möchte.
