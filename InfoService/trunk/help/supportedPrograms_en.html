Supported programs

JAP currenty only supports the following internet protocols:

  • HTTP
  • HTTPS
  • FTP (download via web browser only)
  • Gopher

These are typical programs supported by JAP:

  • Webbrowsers
  • Download-Managers

Therefore, JAP is mainly suitable for anonymous web browsing. The following applications are not supported by JAP:

  • Filesharing, Peer-To-Peer (Emule, BitTorrent, BitComet,...)
  • Chat, Instant Messaging (ICQ, AIM, Yahoo Messenger,...)
  • E-Mail
  • Internet telephony (VoIP, Skype,...)
  • FTP, Telnet, SSH,...

Nevertheless, you find hints on the net that explain how to anonymise the mentioned applications with JAP. They are kind of alchemy and do not work really. Some of these connections, like Skype, may be run over JAP theoretically, but this does not anonymise them, only the traffic is forwarded. The Skype operator himself can deanonymise each of the communication connections with ease, despite the Mixes in the middle.
