Bezahlung: Erweiterte Einstellungen

In diesem Dialog können Sie erweiterte Einstellungen für die Bezahlung vornehmen.

  • Bestätigungsdialog vor der Benutzung eines kostenpflichtigen Dienstes anzeigen
    Wenn Sie diese Option aktivieren, wird ihnen ein Dialog zur Bestätigung angezeigt bevor JAP eine kostenpflichtige Kaskade benutzt. Das hat auch zur Folge, dass JAP sich beim Starten nicht automatisch mit einer Bezahlkaskade verbinden kann.

  • Nicht-anonymisierte Verbindungen zu Bezahlinstanzen erlauben
    Es wird empfohlen, diese Option zu aktivieren. Dadurch kann JAP Informationen über Ihren Kontostand abfragen, auch wenn Sie gerade nicht anonym surfen.
    Wird diese Option nicht aktiviert, können nur über eine anonymisierte Verbindung Kontodaten abgerufen werden.

  • Kontostand automatisch aktualisieren
    Wenn Sie diese Option aktivieren, versucht JAP in regelmäßigen Zeitabständen Ihre Kontoinformationen zu aktualisieren.
