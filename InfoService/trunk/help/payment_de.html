Bezahlung

In diesem Dialog können Sie alle Einstellungen vornehmen, die Sie benötigen, um mit einer kostenpflichtigen Kaskade zu surfen.

Pseudonyme Konten

In der Liste sehen Sie pseudonyme Konten, die Sie bereits angelegt haben. Wenn Sie diesen Dialog zum ersten Mal aufrufen, wird die Liste leer sein. Sie können folgende Aktionen durchführen:

  • Neu:
    Hiermit legen Sie ein neues Konto an. Der Kontoerstellungsprozess kann etwas Zeit in Anspruch nehmen, da hierfür ein aufwändiger kryptographischer Vorgang gestartet werden muss.
    Während des Erstellungsvorgangs werden Sie aufgefordert, eine Sicherheitsfrage zu beantworten. Bitte folgen Sie dazu den Anweisungen am Bildschirm.
    Wenn Ihr Konto erfolgreich erstellt wurde, erscheint es in der Liste.

  • Passwort ändern:
    Durch Klicken dieser Schaltfläche können Sie ein neues Passwort für die Verschlüsselung Ihrer Kontodaten angeben. Das Passwort gilt für alle Konten.

  • Importieren:
    Wenn Sie diese Aktion wählen, können Sie ein zuvor exportiertes Konto wieder importieren.

  • Auswählen:
    Mit diesem Knopf aktivieren Sie das gewählte Konto. Das Konto wird bei der Verbindung zu einer kostenpflichtigen Kaskade zum Bezahlen genutzt.

Informationen zum Kontostand

  • Aufladen:
    Hiermit starten Sie einen Wizard, der Sie durch die Schritte zum Aufladen Ihres Kontos führt.

  • Aktualisieren:
    Wenn Sie diese Aktion wählen, wird der aktuelle Kontostand von der Bezahlinstanz abgefragt.

  • Transaktionen:
    Hier können Sie sich die letzten Aufladetransaktionen zu Ihrem Konto ansehen.

  • Exportieren:
    Dieser Knopf speichert Ihre Kontodaten in einer externen Datei.
    Dies ist sinnvoll, um bei Datenverlust Ihrer Festplatte oder Neuinstallation von JAP oder des Gesamtsystems wieder Zugriff auf Ihr Konto erhalten zu können. Sie sollten die Exportdatei also auf ein externes Medium (z.B. USB-Stick) speichern.
    Es wird empfohlen, ein neu angelegtes Konto zur Sicherung zu exportieren.

  • Löschen:
    Hiermit löschen Sie Ihr Konto.
    ACHTUNG: Sollte sich auf Ihrem Konto noch Guthaben befinden, ist dieses unwiederbringlich verloren!
