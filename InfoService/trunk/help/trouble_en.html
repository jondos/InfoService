Trouble Shooting

Info Service Connection Error

When you try to activate the anonymized access, a pop-up window with the following message appears: Info Service Connection Error.

  • Please check the settings concerning the Info Service. The following values have to be set:

    • Host: anon.inf.tu-dresden.de, Port: 6543. If this is not the case, please correct these settings.
  • Please check if the connection to the web is working properly.
  • Please check if your computer is protected by a firewall. If this is the case, ask your systems administrator to release port no. 6543. If you cannot provide this, unfortunately you cannot use the service currently. For future versions it is planned to direct communication with the first mix via HTTP proxy (by choice). This would solve your problem.
  • If none of the points above fit your problem, it is probably not on your side. In this case we ask you to wait and try again later.
First Mix Connection Error

When you try to activate the anonymized access, a pop-up window with the following message appears: First Mix Connection Error.

  • Please check the settings for the first mix, choose a different cascade or reload the URL of Info Service.
  • Please check if your computer is protected by a firewall. If this is the case, ask your systems administrator to release the port provided in the address of the mix. If you cannot provide this, unfortunately you cannot use the service currently. For future versions it is planned to direct communication with the first mix via HTTP proxy (by choice). This would solve your problem.
  • If none of the points above fit your problem, it is probably not on your side. In this case we ask you to wait and try again later.

Support via E-Mail

If you have a different problem, please try to narrow down the error according to the Debugging Messages on the console. If you are not in the position to do this, please open the JAP web page (http://anon.inf.tu-dresden.de) and take a look if the solution for your problem has already been written down. If this is not the case, please send us an e-mail to the following address: jap@inf.tu-dresden.de

This e-mail should contain the  following information:

  • A detailed description of the error and the situation(s) in which it occurs.
  • A copy of the Error Messages displayed on the console. For achieving this you have to activate all checks concerning debugging in the settings on the register card Miscellaneous Debugging and pull down the debug level to 0.
  • A description of your internet connection (provider, local network, firewall).
  • An e-mail address we can send our reply to.
Please be patient. If many problems occur, the reply will take a certain amount of time.
