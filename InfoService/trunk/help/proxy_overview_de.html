Überblick über verschiedene Proxyarten

Die Wikipedia hat einen Artikel mit einer Auflistung von Proxyarten.
JAP ist ein Anonymitätsproxy, der lokal auf dem Rechner installiert wird. Durch die Nutzung von Mix-Kaskaden wird die Zuordnung von Benutzer zu IP verschleiert, dadurch kann sich der Benutzer anonym im Netz bewegen.
Proxomitron ist ein Filterproxy, der ebenfalls lokal auf dem Rechner läuft. Durch den Einsatz von Filtern können z.B. Werbung aber auch schädliche JavaScripts, Java Applets, Cookies und HTTP Header gefiltert werden. Dazu schreibt Proxomitron die aufgerufene Webseite in Echtzeit um, bevor sie an den Browser weitergeben werden.
