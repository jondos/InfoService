Kommandozeilenargumente für JAP

JAP kann auch über die Kommandozeile gestartet werden. Dabei können ihm verschiedene Befehle mitgegeben werden.
    Befehle (jeweils lange Version, kurze Version):

  • --help, -h

    Zeigt die Hilfe an.

  • --console

    JAP läuft nur in der Konsole, hat also keine Benutzeroberfläche.
    Zum Beenden von JAP tippen Sie "exit" und bestätigen Sie mit Enter.
    Zum Speichern der Konfiguration tippen Sie "save" und bestätigen Sie mit Enter.

  • --minimized, -m

    JAP minimiert sein Fenster direkt beim Start.

  • --noSplash, -s

    Unterdrückt das Wartefenster (Splash-Screen) beim Start von JAP. Der Start wird dadurch etwas länger erscheinen, da es einige Zeit dauert, bis das Hauptfenster von JAP initialisiert ist. Der Spash-Screen ist dazu da, diese Zeit zu überbrücken.

  • --version, -v

    Auf der Konsole werden Versionsinformationen zu JAP und Java ausgegeben.

  • --config {Filename}, -c {Filename}

    JAP wird diese Konfigurationsdatei verwenden.

Beispiel

Um JAP minimiert zu starten, öffnen Sie eine Eingabeaufforderung, wechseln Sie in das Verzeichnis von JAP und geben Sie folgenden Befehl ein:

java -jar JAP.jar -minimized

Sie können diese Argumente auch in Windows-Verknüpfungen verwenden. Klicken Sie dafür einfach mit der rechten Maustaste auf die JAP-Verknüpfung, wählen Sie Einstellungen und editieren Sie die Verknüpfung wie folgt:

Ziel: <Pfad zu JAP>\jap.exe
Ausführen in: <Pfad zu JAP.jar>

Ersetzen Sie dabei <Pfad zu JAP> durch den Pfad, in dem die JAP.jar und die japdll.dll liegen (typischerweise C:\Programme\Jap).
