Zertifikatsstellen

Dieses Panel steht nur in der Expertenansicht zur Verfügung.

Um zu diesem Einstellungsfenster zu kommen, klicken Sie im Hauptfenster von JAP auf Einstellungen und wählen anschließend aus der linken Liste den Eintrag Zertifikatsstellen.

  • Ich will nur von folgenden Stellen zertifizierte Server verwenden:
    Diese Liste enthält alle Zertifizierungsstellen, denen Sie vertrauen. Wird ein Server nicht von einer Stelle, die in dieser Liste enthalten ist, zertifiziert, so wird JAP keine Verbindung mit diesem Server aufbauen.

  • Importieren:
    Mit dieser Schaltfläche fügen Sie Zertifizierungsstellen hinzu. Im erscheinenden Dateidialog muss das entsprechende Zertifikat ausgewählt werden.

  • Entfernen:
    Hiermit löschen Sie den markierten Eintrag aus der Liste.

  • Deaktivieren:
    Durch Klicken dieses Knopfes schalten Sie den markierten Eintrag vorübergehend in den Status "nicht vertrauenswürdig".

  • Details...
    Durch Klicken dieses Knopfes können Sie das Zertifikat des gewählten Eintrages ansehen.
