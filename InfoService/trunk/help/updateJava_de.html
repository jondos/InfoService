Java-Software (JRE) aktualisieren

JAP ist in der Programmiersprache Java geschrieben und benötigt deshalb ein installiertes Java zur Ausführung. Die Installation von Java führt allerdings auch dazu, dass ein sogenanntes Plugin in den Browser integriert wird, mit dem dieser nun Java-Webanwendungen (Applets) ausführen kann. Leider werden auch in diesem Plugin, wie in jeder Software, immer wieder Fehler gefunden, von denen einige auch die Sicherheit des Computers, auf dem diese installiert ist, gefährden können. JAP weist Sie deshalb darauf hin wenn Updates für Ihr installiertes Java vorliegen, und wir empfehlen dringend die Aktualisierung jeweils sofort durchzuführen. Außerdem sollten Sie das Plugin in Ihrem Browser immer deaktivieren, da dessen Ausführung trotz aktivem JAP zur Deanonymisierung Ihrer Internetverbindung führen kann.

Obwohl Java mehrere gleichzeitige Installationen verschiedenen Versionen unterstützt, sollten Sie außerdem immer nur EINE Java-Version auf dem Rechner installiert haben. Deinstallieren Sie deshalb vor einer neuen Java-Installation alle alten Versionen dieser Software.
