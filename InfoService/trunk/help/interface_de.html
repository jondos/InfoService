Die JAP Oberfläche

Anonymität aktivieren
Um anonymes Surfen möglich zu machen muss als erstes der JAP gestartet werden. Danach müssen Sie das Kästchen "Ein" unter "Anonymität" aktivieren. Wenn der Verbindungsaufbau erfolgreich war erscheint kurz danach ein Zeiger im Anonym-O-Meter. Außerdem wird die Nutzerzahl des aktuell gewählten Anonymisierungsdienstes angezeigt.

Hinweis auf notwendige Aktualisierung
Oben rechts im Fenster neben dem JAP-Logo finden Sie einen Hinweis, der Sie zum Aktualisieren der JAP-Software bzw. der Java-Version auffordert. Ein Klick darauf zeigt ihnen, was Sie aktualisieren sollten. Sollte JAP eine Aktualisierung benötigen, öffnet sich dabei das Updatefenster. Im Falle von Java öffnet sich dabei ein Hinweis, daß eine neue Java-Version vorhanden ist. Ein Klick auf "Hilfe" in diesem Popup erklärt Ihnen, wie sie Java aktualisieren können.
Sehen Sie keinen solchen Hinweis, sind sowohl Ihr JAP als auch Java auf dem neuesten Stand.
Rechts daneben sehen Sie die aktuelle Versionsnummer. Ein Klick darauf zeigt Ihnen Informationen zu der Version und den Entwicklern.

Dienstauswahl
Direkt unter dem Logo befindet sich die Auswahlliste für alle verfügbaren Kaskaden, die vom InfoService gemeldet wurden. Wählen Sie einen neuen Dienst aus, verbindet sich JAP mit der neuen Kaskade.
Das Symbol daneben mit den runden Pfeilen aktualisiert die Liste.
Die Schaltfläche "Details" bringt Sie in das Einstellungsfenster von JAP, wo sie Genaueres zu den verfügbaren Diensten erfahren können.
Unter der Auswahlliste für die Dienste finden Sie einen Hinweis, ob die gewählte Kaskade kostenlos oder kostenpflichtig ist.

Anonym-O-Meter
Das Anonym-O-Meter informiert den Nutzer über die aktuelle Auslastung des Anonymisierungsdienstes. Im einzelnen sind dies:

  • Die Anzahl der momentan aktiven Nutzer, die mit dem Internet verbunden sind und im JAP den anonymen Zugriff aktiviert haben.
  • Die Verkehrssituation, d.h. die Menge der Daten, die von dem Anonymisierungsdienst pro Zeiteinheit verarbeitet und weitergeleitet werden.
Aus diesen Informationen kann ein teilweise subjektives Beobachtungsrisiko und eine erreichbare Anonymität abgeleitet werden. Je mehr Nutzer aktiv sind und auch Nachrichten senden bzw. empfangen, um so größer ist die Anonymitätsgruppe jeder einzelnen ins Internet gesendeten Nachricht, da potentiell jeder Nutzer die Nachricht gesendet haben könnte. Nimmt man an, dass der Angreifer eine Anzahl Nutzer kontrolliert bzw. selbst angebliche Nutzer simuliert, so fällt es ihm mit größer werdender Anonyitätsgruppe um so schwerer, alle bis auf einen Nutzer zu kontrollieren. In späteren Versionen soll diese Art Angriff dadurch verhindert werden, dass die Nutzer sich bei den Mixen mit Hilfe sogenannter Tickets, d.h. auf ein Pseudonym ausgestellte digitale Berechtigungsmarken, die mit Hilfe sog. Blinder Digitaler Signaturen gebildet werden, anmelden.

Forwarder

Diese Informationen werden nur in der Expertenansicht angezeigt.

Durch die Forwarding-Funktion können andere JAP-Benutzer über ihr System eine Verbindung zu einem Anonymisierungsdienst aufbauen. Dies ist hilfreich, wenn der Internetprovider oder eine staatliche Hoheit eines JAP-Benutzers den Zugang zum Anonymisierungsdienst unterbindet.
Achtung : Dadurch besteht für Sie die Gefahr in den Verdacht einer Straftat zu geraten, die von einem JAP-Benutzer begangen wurde, der ihr System als Forwarder nutzt.
Folgende Informationen zum Forwarder werden angezeigt:

  • Wieviele weitergeleitete Verbindungen
    • momentan gehalten werden
    • angenommen wurden
    • abgelehnt wurden
  • Wieviel weitergeleitetes Datenvolumen bereits angefallen ist
  • Welche Bandbreite dafür verwendet wird.

Weitere Schaltflächen

  • Die linke untere Schaltfläche lässt JAP auf ein Fenster schrumpfen, das nur noch die wichtigsten Informationen anzeigt. Dieses Fenster dockt automatisch am Bildschirmrand an. Dort können Sie durch einen Klick auf das JAP-Icon die Anonymität ein- oder ausschalten.
    Alternativ können Sie das Minifenster auch aufrufen, wenn Sie im JAP Hauptfenster doppelklicken (nicht jedoch oben im Fensterbalken).
  • Hilfe öffnet die Hilfe zu JAP
  • Assistent hilft Ihnen beim Einrichten von JAP
  • Einstellungen öffnet das Fenster, in dem Sie alle Einstellungen für JAP vornehmen können
  • Beenden trennt die Verbindung zu den Diensten und beendet JAP
