Configuração do JAP  para outros Browsers

É teoricamente possível configurar o JAP para quaquer browser que suporte navegação num HTTP proxy. Apenas tem de observar o menu de configurações e verificar se é possível configurar um servidor proxy e introduzir o seguinte:

  • Endereço de HTTP proxy: 127.0.0.1
  • Porta de HTTP proxy: 4001


Nota: Se configurar uma porta de escuta diferente no JAP, tem de introduzir também aqui essa porta, em vez da 4001.
