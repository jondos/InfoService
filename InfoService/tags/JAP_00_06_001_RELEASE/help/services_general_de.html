Dienste: Allgemein

Dieses Panel steht nur in der Expertenansicht zur Verfügung.

Um zu diesem Einstellungsfenster zu kommen, klicken Sie im Hauptfenster von JAP auf Einstellungen, wählen anschließend aus der linken Liste den Eintrag Dienste und öffnen anschließend den Tab Allgemein.

  • Zuordnung von Internetdiensten zu Anonymisierungsdiensten
    Diese Übersicht zeigt an, welche Internetdienste momentan von welchem Anonymisierungsdienst behandelt werden. In späteren Versionen des JAP werden Sie hier Ihre eigenen Zuordnungen vornehmen können, bisher dient es nur Ihrer Information.

  • Gesamten Datenverkehr ohne Rückfrage blockieren, wenn Anonymitätsmodus aus:
    Ist dieses Feld markiert, leitet JAP nicht-anonymen Verkehr nicht durch. Im Browser erscheint ein Hinweis, dass Surfen momentan nicht möglich ist.
    Aktivieren Sie dieses Feld hingegen nicht, zeigt JAP ein Hinweisfenster mit der Möglichkeit, das nicht-anonyme Surfen kurzzeitig zu erlauben.
    Weitere Informationen

  • Anonymitätsservice sofort nach Programmstart aktivieren:
    Ist dieses Feld markiert, so baut der JAP gleich nach dem Start eine Verbindung zum Anonymisierungsdienst auf.

  • Automatisch wieder verbinden nach Verbindungsabbruch:
    Setzen Sie hier einen Haken, wenn Sie wollen, dass der JAP automatisch die Verbindung zum Anonymiserungsdienst wieder herstellt, wenn diese unterbrochen wurde.

  • Mix-Kaskaden automatisch wechseln
    Bei dieser Option wird beim Start von JAP eine zufällige Kaskade aus allen verfügbaren Kaskaden ausgewählt. Während des Betriebs wird die Kaskade nur dann automatisch gewechselt, wenn die aktive Kaskade ausfällt.

  • Sende Dummy-Daten:
    Hier können Sie festlegen ob und wie häufig Pakete ohne Nutzinhalt verschickt werden, um für einen Beobachter eine laufende Kommunikation vorzutäuschen.

  • Anmelde-Timeout
    Sollten Ihre Internetanbindung zu langsam sein, so kann ein zu geringes Timeout beim Anmelden an den Dienst dazu führen, dass JAP keine anonyme Verbindung aufbauen kann. Ein zu hohes Timeout führt unter Umständen zu hohen Wartezeiten beim Verbinden, wenn der aktuell gewählte Dienst zu langsam oder nicht verfügbar ist. Stellen Sie das Timeout deshalb bei Bedarf höher oder niedriger ein.
