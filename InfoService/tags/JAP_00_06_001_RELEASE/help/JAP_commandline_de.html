Kommandozeilenargumente für JAP

JAP kann auch über die Kommandozeile gestartet werden. Dabei können ihm verschiedene Befehle mitgegeben werden.
    Befehle:

  • -help, --help, -h

    Zeigt die Hilfe an.

  • -console

    JAP läuft nur in der Konsole, hat also keine Benutzeroberfläche.
    Zum Beenden von JAP tippen Sie "exit" und bestätigen Sie mit Enter.
    Zum Speichern der Konfiguration tippen Sie "save" und bestätigen Sie mit Enter.

  • -minimized

    JAP minimiert sein Fenster direkt beim Start.

  • -version

    Auf der Konsole werden Versionsinformationen zu JAP und Java ausgegeben.

  • -config {Filename}

    JAP wird diese Konfigurationsdatei verwenden.

Beispiel

Um JAP minimiert zu starten, öffnen Sie eine Eingabeaufforderung, wechseln Sie in das Verzeichnis von JAP und geben Sie folgenden Befehl ein:
java -jar JAP.jar -minimized
