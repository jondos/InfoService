Proxy-Einstellung (Mac)

Einstellungen bei Apple Macintosh

Wir empfehlen die Verwendung des Internet Explorer. Die Einstellungen für andere Browser ist jedoch sehr ähnlich.

  1. Öffnen Sie den Internet Explorer
  2. Wählen Sie "Bearbeiten:Optionen..."
  3. Im erscheinenden Fenster wählen Sie aus "Netzwerk->Proxies".
  4. Sie machen dann einen Haken bei folgenden Einträgen:
        x Webproxy
        x Gesicherter Proxy
  5. Für "Webproxy" wählen Sie "Einstellungen..." und tragen ein:
        Adresse: 127.0.0.1
        Anschluss: 4001
        Methode: Normal
        ggf. Haken bei "Proxy für lokale Server verwenden" (optional)
  6. Für "Gesicherter Proxy" wählen Sie "Einstellungen..." und tragen ein:
        Adresse: 127.0.0.1
        Anschluss: 4001
        Methode: Tunnel
        ggf. Haken bei "Proxy für lokale Server verwenden" (optional)
