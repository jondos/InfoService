Troubleshooting

InfoService Connection Error

When you try to activate anonymized access, a pop-up window with the following message appears: Error contacting the JAP information service.

  • Please check the settings concerning the InfoService. The following values have to be set: Host: infoservice.inf.tu-dresden.de, Port: 6543. If this is not the case, please correct these settings.
  • Please check if your connection to the internet is working properly.
  • Please check if your computer is protected by a firewall. If this is the case, ask your system administrator to release port 6543. Unfortunately, if you cannot provide this, you cannot use JAP. Direct communication with the InfoService or first mix via HTTP proxy is planned in future versions, which would solve this problem.
  • If none of the points above fit your problem, it is probably not on your side. In this case we ask you to be patient and try again later.

First Mix Connection Error

When you try to activate the anonymized access, a pop-up window with the following message appears: Error connecting the first MIX!.

  • Please check the settings for the first mix, choose a different cascade or reload the URL from the InfoService.
  • Please check if your computer is protected by a firewall. If this is the case, ask your system administrator to release the port provided in the address of the mix. Unfortunately, if you cannot provide this, you cannot use JAP. Direct communication with the InfoService or first mix via HTTP proxy is planned in future versions, which would solve this problem.
  • If none of the points above fit your problem, it is probably not on your side. In this case we ask you to be patient and try again later.

The browser does not work properly or does not use JAP for internet requests

  • Please check whether your browser is properly configured.
  • Please check if you have activated the anonymous access.
  • Please check whether your connection to the internet is working properly.

Support Via E-Mail

If you have a special problem, please try to narrow down the error using the debugging messages on the console. If you are unable to do this, please look at the JAP web page http://anon.inf.tu-dresden.de to see if the solution for your problem has already been mentioned there. If this is not the case, please send us an e-mail to the following address: jap@inf.tu-dresden.de

This e-mail should contain the following information:

  • A detailed description of the error and the situation(s) in which it occurs.
  • A copy of the error messages displayed on the console. to get the appropriate error messages, you have to activate all checks concerning debugging in the Config area under Debugging and set the message level slider to "Debug".
  • A description of your internet connection (provider, local network, firewall).
  • An e-mail address we can send our reply to.

Please be patient. If many people have the same problems, the reply will take a certain amount of time.
