Einstellungen des Browsers / Filters testen

Im folgenden verwenden wir Testseiten, die uns erkennen lassen, ob der Browser (und, soweit vorhanden, der Filterproxy) richtig konfiguriert wurde.
Sie können beide Testseiten anschauen oder sich für eine entscheiden.

Seite 1
Diese Seite gibt einen schnellen Überblick über abgefangene HEADER-Informationen Ihres Browsers. Unter der Überschrift Collected Information können Sie diese Informationen einsehen.
Um erkennen zu können, welche Wirkung die Einstellungen haben, beachten Sie bitte folgendes Vorgehen:

  1. Besuchen Sie die Seite zuerst ohne JAP und Filterproxy bzw. den korrekten Browsereinstellungen. Schreiben Sie die Werte der folgenden Eigenschaften auf (oder merken Sie sie sich):

    1. Reported remote address (zeigt jetzt Ihre externe IP an)
    2. Browser (Die genaue Bezeichnung des Browsers samt Versionsnummer)
    3. OS (Ihr Betriebssystem)
    4. Referer (Die Seite, von der aus Sie auf diese Seite gekommen sind; wird möglicherweise nicht angezeigt)
    Beachten Sie auch, wie viel Information über Ihr System unter dem Punkt Additional Information gesammelt wurde.

  2. Besuchen Sie die Seite nun mit aktiviertem JAP (und Filterproxy).
    1. Sollte nicht die gleiche IP sein wie vorher
    2. Die Daten sollten verfälscht (Firefox -> Opera) oder zumindest deutlich weniger detailliert sein.
    3. Auch hier sollten falsche oder weniger detaillierte Informationen stehen.
    4. Der Referer sollte entweder fehlen oder die aktuelle Seite anzeigen, nicht jedoch die Seite, die sie vorher besucht haben.
    Die zusätzlichen Informationen unter Additional Information sollten fast gänzlich verschwunden sein.
Seite 2
Diese Seite führt ebenfalls Tests durch und gibt dazu Erklärungen ab.
Klicken Sie links im Menü auf Anonymitäts-Test und dann auf den Unterpunkt Zum Anonymitäts-Test. Auch diese Seite sollte man einmal JAP und Filterproxy bzw. den korrekten Browsereinstellungen betrachten und einmal ohne, um den Unterschied sehen zu können.
Besonders bei den folgenden Punkten sollten Unterschiede zu sehen sein:
  • Detaillierte Anzeige Deiner Bildschirmeinstellungen: Hier sollten die angezeigten Werte falsch sein.
  • Anzeige Deiner (öffentlichen) IP-Adresse: Diese Adresse sollte nicht die eigene externe IP-Adresse sein.
  • Anzeige Deiner "echten", privaten IP Adresse trotz NAT an der Firewall: Dieses Java-Applet sollte deaktiviert worden sein. Bei Verwendung von Proxomitron wird ein Hinweis dazu angezeigt.
  • Anzeige des Referers (Referrer): Der Referer sollte, wie dort beschrieben, "http://www.gurusheaven.de/security/anonymitaets_test.shtml" sein. Auch ein nicht vorhandener Referer ist besser als ein "echter", kann aber in seltenen Fällen Probleme bereiten.

Sollten die Testergebnisse nicht den beschriebenen Anforderungen entsprechen, so kann das u.a. folgende Ursachen haben:

  • Es wird kein Filterproxy verwendet.
  • Der Filterproxy verwendet nicht die richtigen Filter.
  • Der Filterproxy benutzt nicht JAP als externen Proxy.
  • Der Browser benutzt nicht den Filterproxy als Proxy.
Weitere Testseiten:
  • WhatIsMyIP Zeigt die externe IP Adresse an.
  • Astalavista Zeigt Verbindungsinformationen an (HTTPS)
  • Proxyjudge Versucht, eine Aussage über das Anonymitätslevel zu machen (irrelevantes Bewertungsschema)
  • RRDB Zeigt Verbindungsinformationen an
  • NetReal Deutsche Seite mit guten Erklärungen
  • Browserspy Enthält eine große Liste mit Identifizierungsmerkmalen
