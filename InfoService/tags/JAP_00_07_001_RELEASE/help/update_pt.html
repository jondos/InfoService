Actualização do JAP

Versão Instalada

  • Versão
    Aqui pode ver o número de versão do JAP que está instalado no seu sistema.

  • Data
    Este campo mostra a data da versão correntemente instalada.

  • Tipo
    O tipo de versão correntemente instalada é mostrado aqui. Há dois tipos possíveis: A "Release Version" que foi intensamente testada e inclui todas as funções que não devem ter erros ou conflitos. A "Development Version" inclui todas as funções disponíveis, mesmo que estejam só parcialmente implementadas ou contenham erros ou enganos.
    Para a maior parte dos utilizadores, recomendamos a "Release Version".

Última Versão

Esta área contém os mesmos campos da  área "Versão Instalada". A informação nesses campos também corresponde aos mesmos campos da área "Versão Instalada". Contudo, a informação não se refere ao software correntemente instalado no seu sistema, mas à versão que pode correntemente ser descarregada da internet. Com a ajuda do menu drop-down "Tipo", pode escolher o tipo de informação a ser mostrada: "Release Version" ou "Development Version".

Info

Este campo mostra informação sobre o progresso da instalação da actualização à medida que é instalada.

Outros Elementos

  • Verificar actualizações
    Clique neste botão para verificar se uma  versão mais recente do que a que tem instalada está disponível para download.

  • Upgrade...
    Com este botão, pode descarregar a última versão do JAP e instalá-la no seu sistema.
