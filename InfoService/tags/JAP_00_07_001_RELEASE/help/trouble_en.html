Troubleshooting

Neither this help nor JAP's built-in assistant (a button in main window) were able to solve my problem

First of all, read the FAQ on our homepage.

I did not find the answer to my question in the FAQ

If you did not find the answer to your problem in the FAQ, you have several possibilities to ask others. But be sure to provide everything of the following information, as otherwise you will get no or an unsatisfying answer:

  • A detailed description of the error and the situation(s) in which it occurs.
  • The version of JAP your are using (appears in the start-up screen and when you click on "i" in the JAP main window).
  • The version of your japdll.dll (appears when you click on "i" in the JAP main window).
  • Your installed Java version(s).
  • The Look&Feel setting of JAP.
  • Name and version of your browser and operating system, ideally information about installed service packs, too.
  • Do you have a desktop firewall? If yes, which one?
  • Do you have an anti virus tool? If yes, which one?
  • A description of your internet connection (your provider, whether you connect via local network, external firewall or forced proxy).
  • A copy of the error messages displayed on the console. To get the appropriate error messages, you have to activate all checks concerning debugging in the config area under Debugging and set all the sliders to the top.

You may post a message to the

User Support Forum

where you will most likely receive answers for your problem quickly. Be sure to use the forum search function before starting a new thread. Most of the questions have already been answered. If this is not the case for your questions, please use a meaningful subject for your posting. Please help other users and us, too, by answering questions in the forum by yourself, if possible.

Only if your problem could not be solved by other users, you should directly contact us via email:

jap@inf.tu-dresden.de

Please understand that in case your question has already been answered somewhere else, we will just direct you to the corresponding answer without providing further instructions. Answering questions will take a certain amount of time, so please be patient.
