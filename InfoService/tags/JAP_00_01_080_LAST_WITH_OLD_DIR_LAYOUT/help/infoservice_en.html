Info Service

General Issues

The Info Service is used for provide the JAP with necessary information on the anonymity service. At the moment this includes:

  • The current JAP version code for updates.
  • The address of the first mix.
  • Information on the mix cascade, e.g. the public keys of the mixes.
  • Information on the serviceís capacity and the level of anonymity achieved by JAP.

Configuration

In the register card Configuration, the IP address of the Info Service is to be entered at Info Service. For changing the IP address, hit the button Change and enter the new address incl. port number.

Please notice that the correct URL is already provided. Changing the URL may disable the JAP or the anonymity service.
